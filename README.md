# RakshakAI 🛡️
### Auto Evidence Capture System

> "Evidence captures itself — so you don't have to."

## 🔍 About
RakshakAI is an AI-powered personal safety system that 
automatically detects suspicious activity and captures 
multi-layer evidence — without any manual action from 
the victim.

## 🚨 Problem
In most attacks (acid attacks, chain snatching, assault),
the victim has no time to react. Evidence is lost.
Attackers escape. Justice is delayed.

## ✅ Our Solution
RakshakAI works in 3 continuous stages:

- **Before** → 10-20 sec background buffer always running
- **During** → Auto-triggers on motion, sound, or SOS
- **After**  → Evidence securely uploaded to cloud

## ⚡ Triggers
- Sudden jerk / shake (Accelerometer)
- Loud sound / scream (Microphone)
- Manual SOS button

## 🔒 What Gets Captured
- Video + Audio recording
- GPS location + Timestamp
- Burst photos
- Emergency alert to contacts

## 🛠️ Tech Stack
- Python Flask (Backend)
- HTML + JavaScript (Frontend)
- Firebase (Cloud Storage)
- Web APIs (Motion, Audio, Geolocation)
